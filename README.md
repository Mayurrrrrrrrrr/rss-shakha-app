# 🚩 संघस्थान पोर्टल (Sanghasthan Portal)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![PHP Version](https://img.shields.io/badge/PHP-7.4%2B-blue.svg)](https://www.php.net/)
[![MySQL Version](https://img.shields.io/badge/MySQL-8.0%2B-orange.svg)](https://www.mysql.com/)

**संघस्थान** एक व्यापक डिजिटल प्रबंधन और सांस्कृतिक पोर्टल है, जिसे राष्ट्रीय स्वयंसेवक संघ की दैनिक शाखा गतिविधियों के प्रबंधन और सार्वजनिक सांस्कृतिक सामग्री के प्रसार के लिए डिज़ाइन किया गया है।

---

## ✨ मुख्य विशेषताएं (Key Features)

### 👥 प्रशासनिक डैशबोर्ड (Admin Dashboard)
- **स्वयंसेवक प्रबंधन**: शाखा के स्वयंसेवकों का डेटाबेस (नाम, पता, आयु, संपर्क)।
- **दैनिक उपस्थिति**: शाखा उपस्थिति दर्ज करना और उसकी रिपोर्ट तैयार करना।
- **गतिविधि प्रबंधन**: सुभाषित, अमृत वचन, गीत और घोषणाओं का प्रबंधन।
- **स्नैपशॉट जनरेटर**: दैनिक रिपोर्ट को सीधे व्हाट्सएप पर साझा करने के लिए JPG इमेज बनाना।

### 🚩 सार्वजनिक सांस्कृतिक पोर्टल (Public Portal)
- **तिथि कैलेंडर**: विक्रम संवत और शालिवाहन शक के साथ एक इंटरैक्टिव पंचांग।
- **सांस्कृतिक संग्रह**: सुभाषित, गीत, घोषणाओं और अमृत वचनों का विस्तृत आर्काइव।
- **एकात्मता स्तोत्र**: एक इंटरैक्टिव "पाण्डुलिपि" शैली में एकात्मता स्तोत्र का अर्थ सहित अध्ययन।
- **खोज सुविधा**: पूरे संग्रह में किसी भी विषय को खोजने की क्षमता।

## 🛠️ तकनीकी संरचना (Technical Architecture)

- **Backend**: PHP (Vanilla) + PDO (Secure DB access)
- **Database**: MySQL 8.0+
- **Frontend**: HTML5, CSS3 (Modern Responsive Design), JavaScript (Vanilla)
- **Design System**: "Indian-Minimalist" - केसरिया और स्वर्ण रंगों का उपयोग।

## 📂 फ़ाइल संरचना (Directory Structure)

```text
rss-shakha-app/
├── assets/             # CSS, JS, और इमेजेज
├── includes/           # साझा हेडर, फुटर और ऑथेंटिकेशन
├── config/             # डेटाबेस और सिस्टम कॉन्फ़िगरेशन

├── amrit-vachan/       # सार्वजनिक अमृत वचन संग्रह
├── ekatmata-stotra/    # इंटरैक्टिव एकात्मता स्तोत्र

└── index.php           # लॉगिन पृष्ठ
```

## 🚀 स्थापना (Installation)

### 1. प्रोजेक्ट क्लोन करें:
```bash
git clone https://github.com/Mayurrrrrrrrrr/rss-shakha-app.git
```

### 2. डेटाबेस सेटअप:
- `sql/schema.sql` को अपने MySQL सर्वर में इम्पोर्ट करें।
- `.env.example` को `.env` के रूप में कॉपी करें और अपनी DB जानकारी भरें।

### 3. सर्वर कॉन्फ़िगरेशन:
- सुनिश्चित करें कि `mod_rewrite` इनेबल्ड है (SEO-friendly URLs के लिए)।
- PHP GD लाइब्रेरी (इमेज जनरेशन के लिए) इंस्टॉल करें।
