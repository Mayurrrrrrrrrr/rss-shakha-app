# 🚩 शाखा प्रबंधन — RSS Shakha Management App

दैनिक शाखा गतिविधि एवं उपस्थिति प्रबंधन वेब ऐप

## ✨ विशेषताएँ

- 🔐 **लॉगिन सुरक्षा** — बिना लॉगिन कोई एक्सेस नहीं
- 👥 **स्वयंसेवक प्रबंधन** — नाम, पता, फ़ोन, उम्र
- 📋 **गतिविधि प्रबंधन** — डिफ़ॉल्ट + कस्टम गतिविधियाँ
- 📝 **दैनिक रिकॉर्ड** — उपस्थिति, गतिविधि चेकलिस्ट, संचालक, संदेश
- 📄 **सूची दृश्य** — सभी रिकॉर्ड की सूची
- 📅 **कैलेंडर दृश्य** — महीने के अनुसार रिकॉर्ड
- 📸 **स्नैपशॉट (JPG)** — रिपोर्ट की तस्वीर बनाएँ
- 📱 **व्हाट्सएप शेयरिंग** — सीधे मोबाइल से भेजें

## 🛠️ WSL पर सेटअप

### 1. WSL टर्मिनल खोलें और प्रोजेक्ट फ़ोल्डर में जाएँ:
```bash
cd /mnt/c/Users/mayur/.gemini/antigravity/scratch/rss-shakha-app
```

### 2. सेटअप स्क्रिप्ट चलाएँ:
```bash
chmod +x setup_wsl.sh
./setup_wsl.sh
```

### 3. ब्राउज़र में खोलें:
```
http://localhost/rss-shakha-app/
```

### 4. लॉगिन करें:
Set your credentials during first-run setup. See SETUP.md.

## ⚠️ ध्यान दें

- पासवर्ड पहले लॉगिन के बाद बदल लें
- स्नैपशॉट के लिए PHP GD एक्सटेंशन आवश्यक है
- हिंदी फ़ॉन्ट (`fonts-noto-core`) स्नैपशॉट में हिंदी दिखाने के लिए ज़रूरी है
- मोबाइल पर व्हाट्सएप शेयरिंग Web Share API का उपयोग करती है
